# Product Requirements

## Overview

The AI Asset Management Assistant helps tenants, unit owners, HOA/board members, building owners, and property managers ask property-related questions without exposing data outside their permission scope.

## Roles

- Tenant
- Unit Owner
- HOA / Board Member
- Building Owner
- Property Manager / Admin

## MVP Use Cases

- Tenants check rent, due dates, lease facts, and maintenance status.
- Unit owners review unit income, tenant payment status, and maintenance issues.
- Board members review building-level operational issues without private tenant data.
- Building owners review portfolio performance, occupancy, risk, and overdue totals.
- Property managers review overdue payments, escalations, unresolved maintenance, conversations, and audit logs.

## Success Metrics

- Reduced repetitive calls and emails.
- Clear role-specific answer differences.
- Every chat turn auditable.
- Unauthorized data requests blocked.
- Escalations created when AI cannot safely answer.
